# Panter Hotel

A Flask web application for hotel room reservations built with Python, Flask, and SQLite.

## Features
- Welcome page with Book a Room and Manager options
- Room reservation form with name, check-in, check-out, and room type
- Confirmation page after booking
- Manager view to see all reservations in a table

## How to Run
1. Install Flask: pip install flask
2. Run: python3 app.py
3. Open browser at http://127.0.0.1:5000

## Tech Stack
- Python 3
- Flask
- SQLite
- Bootstrap 5
